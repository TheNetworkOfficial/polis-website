FAQ

Frequently asked questions

Quick answers on permissions, safety practices, events, and account controls.

Is location required to use Polis?

No. Users can set home area manually and can choose not to grant GPS permissions. Location-enabled experiences are optional.

What data categories does Polis handle?

High-level categories include account/profile identifiers, user content (posts/comments/events), reports, optional location data, diagnostics, and abuse-prevention signals.

See Privacy Policy and Data Safety for details.

How do reporting and moderation work?

Users can report content in-app and can block/mute users. Moderation actions may include content restriction and account enforcement where needed.

Are paid events currently supported?

Free event signup is live. Paid event processing is currently a coming soon path and should not be interpreted as active checkout support.

How do I delete my account?

You can start deletion from in-app settings or use the web Delete Account page.

Deletion is irreversible once executed. If a request is pending, access may be restricted during processing.
